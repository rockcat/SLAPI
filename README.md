# SLAPI
Serial Line API Protocol

A proxy protocol for slow/retro devices to access web APIs

See [SLAPI Protocol v1.0](https://github.com/rockcat/SLAPI/blob/main/SLAPI%20Protocol%20v1.0.md) for details of the protocol

The lib folder has an implementation example for the RPi Pico 2W in MicroPython
Explanation of the [GPIO implementation](https://github.com/rockcat/SLAPI/blob/main/GPIO%20protocol.md)

Example client in BASIC in the cpm folder
Z80 assembly coming soon...
