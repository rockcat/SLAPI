# SLAPI
Serial Line API Protocol

A proxy protocol for slow/retro devices to access web APIs

See [SLAPI Protocol v1.0](https://github.com/rockcat/SLAPI/blob/main/SLAPI%20Protocol%20v1.0.md) for details

The lib folder has an implmentation example for the RPi Pico 2W in MicroPython

Example client in BASIC in the cpm folder
